# CipherLab
Educational cryptography and cryptanalysis toolkit (Playfair + DES)

CipherLab is an interactive cryptography and cryptanalysis laboratory built in Python.
It allows users to explore encryption, decryption, and real-world cryptanalytic attacks through a clean, intuitive graphical interface.

This toolkit combines classical cryptography with modern attack techniques, making it ideal for cybersecurity students, cryptography learners, and academic demonstrations.
