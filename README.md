# CipherLab
Educational cryptography and cryptanalysis toolkit (Playfair + DES)

CipherLab is an interactive cryptography and cryptanalysis toolkit built in Python.
It provides a graphical environment for experimenting with classical and modern cryptographic techniques, as well as automated attacks for educational and demonstration purposes.

Features

Playfair cipher encryption and decryption

DES encryption and decryption

Dictionary-based and crib-based key attacks

Automated Playfair key recovery using hill-climbing optimization

File import support for ciphertexts and keys

Multithreaded attack execution with progress monitoring

Clean, user-friendly graphical interface using Tkinter

System Requirements

Python 3.9 or later

Operating System: Windows / Linux / macOS

Installation

Clone the repository and install the required dependency:

pip install -r requirements.txt

Running the Application

From the project directory:

python cyberSecDemo.py

Educational Use

This project is intended for academic learning, cryptography laboratories, and cybersecurity demonstrations.
It illustrates classical cipher construction, symmetric encryption, heuristic cryptanalysis, and secure software design principles.
